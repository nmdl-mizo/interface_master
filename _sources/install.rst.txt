Installation
###########################################

Installing interface master
===========================================
::

    pip install git+https://github.com/nmdl-mizo/interface_master

or

::

    git clone https://github.com/nmdl-mizo/interface_master
    cd interface_master
    pip install .

Requirements
===========================================

- python3
- pymatgen
- matplotlib
- gb_code
- numpy (requirements already satisfied by installing pymatgen)
